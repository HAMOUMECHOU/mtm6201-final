# NutriLife - Healthy Diet Planning Website

## Project Overview
A responsive nutritionist website built with Bootstrap, featuring personalized meal plans and nutrition coaching services.

## Pages Included
- **Home** (`index.html`) - Hero section and service overview
- **About** (`about.html`) - Team information and mission
- **Services** (`services.html`) - Pricing plans and process

## Technologies Used
- HTML5
- CSS 
- Bootstrap 
- Font Awesome
- Google Fonts

## Features
- Responsive design for all screen sizes
- Accessibility compliant (ARIA roles, skip links)
- Custom color scheme using CSS variables
- Image optimization with picture element
- CSS animations and hover effects

## Assets
All images and design elements are original.
